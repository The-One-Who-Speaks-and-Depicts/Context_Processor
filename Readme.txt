Описание:

Программа предназначена для исследователей, пользующихся методом контекстного анализа. Её задача - автоматизировать процесс обработки
контекстов, семантический и квантитативный. Программа позволяет для заданной лексемы дать количество встретившихся контекстов в корпусе,
задать как минимум один контекст,  дать общесемантический анализ и специализированный семантический анализ по критерию, заданному 
исследователем в поле "Основание". Данные получают html-разметку, способствующую облегчению восприятия и доступную для переноса в 
.doc/docx-файл. Также оформляются ссылки на источник, откуда взят контекст.


Для установки:
1) Распакуйте архив WordProcessor.exe.
2) В появившейся папке запустите файл WordProcessorTrial.Application.
3) Следуйте инструкциям.
4) Пользуйтесь файлом WordProcessorTrial.Application для запуска.

Использование:
1) Поочерёдно внесите значения в каждое из полей сверху вниз и слева направо, нажимая кнопку "Ввести" после окончательного заполнения поля.
Результат ваших действий будет виден в поле "Итог".
2) Когда все поля будут заполнены, нажмите активировавшуюся кнопку "Внести в файл".
3) Создайте файл, выберите новый либо сохраните в созданный в этот же сеанс. Рекомендуется создание файла с расширением .html, его открытие
и дальнейшее копирование  из него в файл с расширением .doc/.docx.
